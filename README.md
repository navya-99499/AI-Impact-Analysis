# 🤖 AI Impact Analysis

## 📌 Overview
This project explores the impact of Artificial Intelligence across different domains using data-driven analysis.

The goal is to understand how AI influences trends, adoption patterns, and key metrics across industries through exploratory data analysis and visualization.

---

## 📊 Dataset
- **Source:** AI-related dataset (industry / trend analysis)
- **Dataset Link:**https://www.kaggle.com/datasets/sahilislam007/ai-impact-on-job-market-20242030?utm_source

---

## 🛠️ Data Preprocessing
The dataset was prepared using the following steps:

- Handling missing values
- Data cleaning and formatting
- Feature selection for analysis
- Preparing data for visualization and insights

---

## 🔍 Exploratory Data Analysis
The analysis focuses on:

- AI adoption trends across industries
- Distribution of AI usage
- Comparative analysis across categories
- Identifying key influencing factors

---

## 📈 Key Insights
- Identified major domains where AI adoption is growing rapidly
- Analyzed trends and variations across different categories
- Observed patterns in AI impact across sectors
